# Riggleman Sales Coach - Quick Start (30 seconds)

## 1. Add Your OpenAI API Key

Edit `.env.local`:
```
OPENAI_API_KEY=sk-your-key-here
```

## 2. Start Development Server

```bash
npm run dev
```

## 3. Open in Browser

Go to: http://localhost:3000

## 4. Use the App

1. Click "Start Coaching Session"
2. Click the microphone button
3. Speak about your sales challenges
4. Listen to the AI coach's response
5. See the conversation on the right

## Done! 🎉

That's it. The app is fully functional with:
- Voice input (Whisper)
- AI coaching (GPT-4)
- Audio responses (TTS)
- Transcript display
- Dark professional UI

## Deployment

When ready to deploy:

```bash
# Option 1: Vercel (one-click)
npm i -g vercel
vercel

# Option 2: Production build
npm run build
npm run start
```
