🎯 PROJECT DESCRIPTION

An AI-powered voice coaching application for Hyundai and Kia sales
professionals. Users have natural conversations with an AI coach who
provides actionable advice on sales performance, call handling, objection
management, and closing techniques.

✨ KEY FEATURES

Voice Chat
  • Microphone input with Web Audio API
  • Real-time recording
  • Automatic transcription
  • Natural conversation flow

AI Coaching
  • Hyundai/Kia sales specialist
  • Asks about metrics (call volume, conversion, objections)
  • Provides actionable advice
  • Maintains conversation context

Professional UI
  • Dark theme (slate-900)
  • Green accents (emerald-600)
  • Responsive layout
  • Real-time transcript display

Production Ready
  • Full TypeScript support
  • Error handling
  • Environmental configuration
  • Vercel deployment ready

🚀 QUICK START

1. Configure API Key
   Edit .env.local and add: OPENAI_API_KEY=sk-...

2. Start Development Server
   npm run dev

3. Open Browser
   http://localhost:3000

4. Test the App
   • Click "Start Coaching Session"
   • Click microphone and speak
   • Listen to coach response
